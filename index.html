IOT-Advanced

The project that we have been working on was a group project, which was completed by Brecht Voets, Roel Metdepenningen, Harold De Ridder, Jonas De Weerdt, and myself. The project was a part of the IOT Advanced course, which was taught by Tom Van den Broeck. This project taught us how to work together in a team and how to delegate tasks when necessary.

My responsibilities in the project were diverse. I was responsible for creating the documentation and presentation for the project. I also helped with connecting the LCD, creating the local online dashboard, and integrating the data with influxdb. I also assisted with the push notifications because they went through my dashboard. Additionally, I was responsible for time management and communication within the team.

The system that we developed was designed to monitor and analyze various electrical parameters for a single team. The system measured all electrical parameters in 1-minute intervals, including current consumption/injection (for all phases), total consumption/injection for day and night tariffs (for all phases), and other parameters in 15-minute intervals, such as total gas consumption and peak detection for the current 15-minute interval.

The system also included an alarm function that notified users via push notification if a pre-defined threshold was reached. There was also an output contact (relay) for load management.

Overall, this project was a great learning experience, and we were able to develop a functional system that could help users monitor and manage their electrical parameters more efficiently and effectively. The project taught us the importance of teamwork, communication, and delegation of tasks, which are essential skills for any project.
